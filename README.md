# sm-todo-backend
Learning App

## Requirements
 Node 14

## Getting started

Clone the repository
```
git clone git@github.com:marekke/sm-todo-backend.git
```

Switch to the repo folder
```
cd sm-todo-bkacned
```

Install all dependencties
```
npm install
```

Run the app
```
npm start
```

Server default is listening on 8080 port


Call your endpoint  
http://localhost:8080/api/todos
